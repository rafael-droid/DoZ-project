DoZ-project
Projekt wspomaga działanie apteki. Zaimplementowane funkcjonalności to:
        - wprowadzanie cenników
        - pobieranie zestawień sprzedaży 
Do tworzenia projektu została wykorzystana baza danych MySQL.
